# QuickLink – URL Shortener System

A console-based URL shortening system built using Java.

## Features
- URL shortening
- Custom aliases
- Expiry dates
- Click analytics
- Search functionality
- Soft delete / hard delete
- CSV-based persistence

## Technologies Used
- Java
- File I/O
- OOP
- Collections Framework

## Run

```bash
javac -encoding UTF-8 src\quicklink\*.java
java -cp src quicklink.Main
